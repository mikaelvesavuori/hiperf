Welcome to this example page using hiperf. The primary font used on the page is Libre Franklin by Impallari Type, which is taken from Google Fonts. All links are using rel="nofollow" which is a way of closing a well-known security hole.

Below we will see four lazyloaded images from Unsplash Source and its users. I highly recommend Unsplash when in need of imagery. Also, for accessibility, don't forget to set title and alt tags for all images!

Lazyloaded image with responsive markup. Uses a blank base64 data:image/gif to render empty content on first load.

Lazyloaded images will begin loading when your viewport is nearing the image.

Use a component to make it easier to write all the code needed for this kind of image markup. You will save a lot of time.

The image transition coming from the side (more specifically, a transform) is done with transform: translateX(0) and will-change: transform so that the user's GPU will be utilized, resulting in more performative animations.

The images get a lazyload class appended dynamically on page load. The syntax is modern ES6, but we are also preloading a tiny, smart and dynamic polyfill to detect what our browser is missing in order to fully support ES6. This means you no longer need to be concerned about compiling legacy ES5 code. Of course, always test, but the service – Polyfill.io – is really very good.
